Slack Directory listing screenshots

Six 1600×800 slides for Echo-Synch's Slack Directory listing carousel. Each is a self-contained HTML file that renders to a pixel-perfect PNG when captured at 1600×800.

How to capture each as a PNG

  1. Open the slide HTML in Chrome (links below).
  2. Open DevTools (F12) → toggle device toolbar (Ctrl+Shift+M).
  3. Set custom dimensions: 1600 × 800. Zoom should be 100%.
  4. Hit Ctrl+Shift+P → type "Capture screenshot" → Enter. Saves a 1600×800 PNG.
  5. Upload to Slack Directory submission. Repeat for each slide.
