Configuration Guide

EcoSynch is designed to be "Zero-Configuration." However, for the bot to work effectively, your Jira and Confluence instances must be properly linked. Follow these three steps.

Step 1: Installation

  1. Go to the Atlassian Marketplace.
  2. Search for EcoSynch.
  3. Click "Get it now" and select your site.
  4. Wait for the "Installation Successful" notification.

Step 2: Linking Applications

EcoSynch relies on the standard application link between Jira and Confluence.

  1. Open any Jira Issue.
  2. Look for the "Link issue" button (usually below the title).
  3. Select "Confluence Page".
  4. Search for and select the relevant documentation page.

Note: You do not need to configure API keys. EcoSynch inherits permissions automatically via the Atlassian Connect framework.

Step 3: Workflow Trigger

The bot wakes up when a ticket status changes to Done.

  • Ensure your Jira Board columns are mapped correctly.
  • Dragging a ticket to the right-most column (Done/Released) will trigger the sync.
  • Allow 10-30 seconds for the AI to process the text and update Confluence.
