Trust Center

Encryption

All data is encrypted in transit via TLS 1.2+ and at rest using AWS KMS (AES-256). API Keys are stored in a dedicated Secrets Manager vault.

Isolation

EcoSynch uses a multi-tenant architecture with strict logical isolation. Tenant IDs are enforced at the database and application level.

Least Privilege

Our Atlassian scopes are strictly limited. We request read:jira-work and write:confluence-content only. We cannot read your user directory or admin settings.
